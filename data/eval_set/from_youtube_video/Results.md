# Benchmark results

California P&C broker-style **multiple-choice** eval (YouTube / transcript track): `eval_set/from_youtube_video/questions.txt` + `eval_set/from_youtube_video/answers.txt`. Most runs are **questions-only** (option + short reason) in `results/*_reasoned.csv`. **`gpt-5.5`** uses a **two-phase** protocol (stem-only reasoning, then options) — see `results/openai_gpt-5.5_two_phase.csv`.
